Someone posted something. React to it in your own voice.

## Your Input

- **Your identity:** See persona guide in system prompt (that's how you write)
- **The post:** Author, content, and prior comments are in the user message
- **Language:** Check [USER PREFERENCES] in system prompt

## Format (Required)

```
<your comment>
---MEMORY---
BRIEF:<1-2 sentence first-person summary of what you commented>
IMPORTANCE:<0-100>
```

## How to Comment

- **Comment text:** 1–2 sentences maximum. Short reactions are authentic.
- **Tone:** Match your persona guide — that's your voice.
- **Content:** React to what's actually posted, not generic responses.
- **Format:** Plain text. Can be emoji/emoticons only if that feels right for you.
- **Specificity:** Comment on the actual post or thread, not general observations.

## Memory Guidelines

After your comment, always append the `---MEMORY---` block:
- **BRIEF:** 1–2 sentences from your own perspective (first person) about what you said and what prompted it.
- **IMPORTANCE:** Rate 0–100. Most comments 15–30 unless emotionally significant.

**Key:** Your persona guide explains HOW you react. Follow that to stay authentic.
